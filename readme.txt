	Description
	
	This project is the simply basketball simulation with augmented reality technology.
	It has developed for IOS 12.0, for iPhone devices and portrait orientation.
	It has implemented two new Apple frameworks: SceneKit and ARKit.
	This application is simulating basketball game. It has four buttons: add hoop button, hoop horizontal moving button, hoop round moving button, stop moving button. After tapping add hoop button basketball backboard with hoop will appear on screen. By tapping on screen ball will appear and throw to the hoop. The user should manage the ball with changing phone position in space. Also the hoop can moving in two directions: horizontal (four times) and round (two times) after tapping on proper buttons. If user wants stopping hoop moving immediately, he should press stop button. It will getting more difficulty to throw the ball into the basket.
